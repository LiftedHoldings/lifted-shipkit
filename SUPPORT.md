# Support

Need help with ShipKit? Here is where to go, fastest path first.

## Documentation

Most questions are answered in the docs:

- [Quickstart](docs/quickstart.md) — install, configure, run.
- [Integration guide](docs/integration.md) — drop-in JS widget, config table,
  callbacks, framework snippets.
- [Managed tier](docs/managed.md) — the plug-and-play hosted widget and how to
  get a managed key.
- [Authentication](docs/authentication.md) — minting API keys and the `ShipKit-Api-Key` header.
- [Architecture](docs/architecture.md) — how the pieces fit together.
- [API reference](docs/api.md) — backend endpoints.
- [3-D Secure explained](docs/3d-secure.md) — the payment security model.

## Questions and ideas — GitHub Discussions

For "how do I…", design questions, integration help, and feature ideas, open a
thread in **GitHub Discussions**. It is the best place to reach maintainers and
other developers, and answers help everyone who searches later.

Before posting, a quick search of existing Discussions and issues often turns up
an answer immediately.

## Developer help — email

For direct developer assistance — integration questions, deployment help, or
anything about the managed tier — email **support@liftedholdings.com**.

When you write in, include:

- What you are trying to do.
- What you expected vs. what happened.
- Your deployment mode (self-host or ShipKit Managed) and version.
- Relevant logs — **with all secrets removed** (no API keys, tokens, merchant
  IDs, or credentials).

## Bugs and feature requests

- Found a bug? Open an issue with the **Bug report** template.
- Want a feature? Open an issue with the **Feature request** template.

See [CONTRIBUTING.md](CONTRIBUTING.md) if you would like to fix it yourself — we
welcome pull requests.

## Security issues

Do **not** use Discussions, public issues, or email support for vulnerabilities.
Follow the private disclosure process in [SECURITY.md](SECURITY.md).

## Merchant accounts and payments

ShipKit's payment layer is secured by **Lifted Payments 3-D Secure**. To apply
for a 3-D Secure merchant account or get a managed key, visit
[liftedholdings.com/payments](https://liftedholdings.com/payments).
